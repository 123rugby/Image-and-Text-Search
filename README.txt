For linux users:

Download Pycocotools (not necessary for testing)
git clone https://github.com/pdollar/coco.git
Run 'make install' in the directory PythonApi


Install gensim
pip install --upgrade gensim


Install Caffe by following these tutorials:
http://caffe.berkeleyvision.org/install_apt.html
http://installing-caffe-the-right-way.wikidot.com/start
Don't forget to make pycaffe and to have caffee in the repository
And then download googlenet neural network
